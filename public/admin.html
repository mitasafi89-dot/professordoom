Skills — working contracts

Each skill is a guideline document the reviewer treats as a binding working contract. Pick a skill, upload its document (.docx, .pdf, .txt, .md) or paste the text, and save. Users pick a skill in chat; with none selected, it’s a normal chat.

 
The database connection is set once via the DATABASE_URL environment variable and is not editable here. Refresh tokens can be revoked by signing out of your account. Sending each message still requires solving the bot-verification widgets in the composer.
